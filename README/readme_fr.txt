--- INSTALLATION ---

0)
L'utilisation desktop d'Open Sankey nécessite :
- un serveur web local,
- php v.5 ou ultérieure. Dans la configuration de php les "magic quotes" doivent être activés.

1)
Le dossier "open-sankey" doit être placé dans un répertoire du serveur web local. Par exemple, sur macos x, dans un sous-répertoire de "~/sites".

2)
Vérifier que les dossiers situés dans "sources/user_sankey" sont en lecture/écriture (le serveur doit pouvoir créer des fichiers dedans).   


--- UTILISATION ---

Pour lancer l'application, ouvrir l'url correspondant au fichier "open-sankey/index.php" dans un navigateur. 
Par exemple, sur macos x : "http://localhost/~user/open-sankey/index.php".
L'application n'est a priori pas compatible avec Internet Explorer.
